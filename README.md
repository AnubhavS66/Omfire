# 🔥 Om Fire - 2D Battle Royale 
**Om Fire** is an Indian-themed 2D top-down action game.

### 🎮 Features:
- **5-Stage Navigation:** Splash, Lobby, Selection, Gameplay, Result.
- **Combat:** Projectile-based shooting (Agniban).
- **Abilities:** Unique power-ups for Agni, Vayu, Moksha, and GLW.

### 🚀 Technical Info:
- Engine: Max2D (Logic Nodes + JavaScript)
- Input: Dual-joystick / Touch Buttons
