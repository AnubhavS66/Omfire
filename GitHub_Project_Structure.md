# 📂 Om Fire - File Structure

- `/Assets`: Saare sprites (Agni, Vayu, Desert Map).
- `/Scripts/CombatSystem.js`: Goli aur Damage ka logic.
- `/Scripts/Movement.js`: Joystick aur Player ki speed.
- `/Scenes/`: 
    1. **Splash**: Logo & Loading.
    2. **Lobby**: Start/Character Select.
    3. **Battlefield**: Asli game map.
    4. **Result**: Winner/Loser screen.
- `ProjectConfig.json`: Global variables (HP, Ammo, Kills).
